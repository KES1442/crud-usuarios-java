# CRUD de Usuários em Java

Projeto simples de CRUD de usuários desenvolvido em Java, com foco em treinar lógica de programação, estruturas de dados e boas práticas.

## Funcionalidades
- Criar usuário
- Listar usuários
- Atualizar usuário
- Deletar usuário

## Tecnologias utilizadas
- Java 21 LTS
- ArrayList
- Iterator
- Scanner

## Estrutura do projeto
src/
 ├── CadastroUser
 ├── User
 └── Test

## Aprendizados
- Lógica de programação
- Estruturas de dados
- Uso de Iterator para evitar ConcurrentModificationException
- Organização de código em pacotes

## Como executar
Execute a classe `CadastrarUser`.
